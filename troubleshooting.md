# Troubleshooting (English)

> Common symptoms and how to handle them.

---

<table><thead><tr><th width="280">증상</th><th>대응</th></tr></thead><tbody><tr><td>skill이 패널에 안 보임</td><td>패널 새로고침 / 앱 재시작</td></tr><tr><td>로컬 파일을 못 찾음</td><td>채팅 입력창의 <strong>+ → Quick knowledge → Choose a folder</strong> 로 폴더 다시 지정</td></tr><tr><td>Space/Agent 생성 메뉴가 없음</td><td><strong>정상</strong> (Desktop은 사용만, 생성은 Web)</td></tr><tr><td>응답이 너무 짧음/얕음</td><td><strong>Smart 모드</strong>로 전환 (필요시 Thinking 토글)</td></tr><tr><td>Connection 인증 실패</td><td>회사 계정으로 재인증 (OAuth 90일 만료)</td></tr><tr><td>외부 도구 데이터가 안 옴</td><td>Settings에서 연결 확인, 권한 허용</td></tr></tbody></table>

---

<table><thead><tr><th width="280">Symptom</th><th>What to do</th></tr></thead><tbody><tr><td>Skill doesn't show up in the panel</td><td>Refresh the panel or restart the app</td></tr><tr><td>Can't find local files</td><td>Re-select the folder via <strong>+ → Quick knowledge → Choose a folder</strong> in the chat input</td></tr><tr><td>No Space/Agent creation menu</td><td><strong>Expected</strong> — Desktop is for usage only; creation happens on the Web</td></tr><tr><td>Responses are too short / shallow</td><td>Switch to <strong>Smart mode</strong> (turn on Thinking toggle if needed)</td></tr><tr><td>Connection auth failure</td><td>Re-authenticate with your corporate account (OAuth expires every 90 days)</td></tr><tr><td>No data from external tool</td><td>Check the connection in Settings and grant permissions</td></tr></tbody></table>
