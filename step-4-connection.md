# STEP 4. Connection — 외부 도구 연결

> Outlook, SharePoint 같은 회사 도구를 Quick과 연결해서 내 캘린더·이메일·문서를 채팅으로 다룹니다.

---

## ① 연결 열기

**Settings → Capabilities → Connectors** 를 엽니다.

## ② 도구 선택

**Outlook** 또는 **SharePoint** 를 선택합니다.

## ③ OAuth 로그인

회사 계정으로 로그인합니다.

## ④ 동작 확인

```
이번 주 내 캘린더 일정 요약해줘.
```

또는

```
[SharePoint 폴더명]에서 최근 문서 3개 찾아서 핵심만 정리해줘.
```

→ 권한 요청 뜨면 허용.
